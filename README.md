# 🏋️ FitTrack - Fitness Tracking App

A modern full-stack fitness tracking application where users can track calories, activities, and fitness goals.

---

## 🚀 Live Demo
👉 https://fit-track-hazel.vercel.app/

---

## ✨ Features

- 🔐 User Authentication (JWT)
- 🍽️ Food Logging & Calorie Tracking
- 🏃 Activity Tracking
- 🎯 Personalized Goals
- 🌗 Dark / Light Mode
- 📱 Responsive UI
- 🤖 AI Food Detection

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios
- React Router

### Backend
- Strapi (Headless CMS)
- JWT Authentication
